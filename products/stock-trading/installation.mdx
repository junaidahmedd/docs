---
title: 'Installation'
description: 'Install Revo Stock Trading and set it up on your FiveM server.'
---

<Steps>
  <Step title="Download Stock Trading">
    Visit the <a href="https://keymaster.fivem.net/" target="_blank">FiveM Keymaster</a> and download the latest version of **Revo Stock Trading**. Save the script to your computer.
  </Step>
  <Step title="Add to Resources">
    Place the downloaded <code>revo_stocktrading</code> folder inside your server's <code>resources</code> directory.
  </Step>
  <Step title="Import Required SQL Files">
    Import the following SQL files into your game database (using your preferred MySQL client):
    <ul>
      <li><code>stocks.sql</code></li>
      <li><code>trade_users.sql</code></li>
      <li><code>trades.sql</code></li>
    </ul>
    Make sure the database configured for <code>oxmysql</code> matches where you import these tables.
  </Step>
  <Step title="Configure server.cfg">
    Ensure dependencies and the resource are started in the correct order:
    ```bash
      ensure oxmysql
      ensure revo_stocktrading
    ```
  </Step>
  <Step title="Add Trading Tablet to Your Inventory System">
    <Tabs>
      <Tab title="ox_inventory">
        Open your <code>ox_inventory/data/items.lua</code> file and add:
        ```lua
          ['revo_trading_tablet'] = {
            label = 'Trading Tablet',
            weight = 500,
            stack = false,
            close = true,
            description = 'Access trading panel.',
            client = {
              image = 'REVO_TRADING_TABLET.png',
            }
          }
        ```
        <Tip>
          Copy <code>REVO_TRADING_TABLET.png</code> from the script's <b>inventory item</b> folder to <code>ox_inventory/web/images</code>.
        </Tip>
      </Tab>
      <Tab title="qs-inventory">
        Open your <code>qs-inventory/html/images</code> folder and add the image.<br/>
        Then add the item to your <code>shared/items.lua</code>:
        ```lua
          ['revo_trading_tablet'] = {
            name = 'revo_trading_tablet',
            label = 'Trading Tablet',
            weight = 500,
            type = 'item',
            image = 'REVO_TRADING_TABLET.png',
            unique = false,
            useable = true,
            shouldClose = true,
            combinable = nil,
            description = 'Access trading panel.'
          }
        ```
        <Tip>
          Place <code>REVO_TRADING_TABLET.png</code> in <code>qs-inventory/html/images</code>.
        </Tip>
      </Tab>
      <Tab title="qb-inventory">
        Open your <code>qb-inventory/html/images</code> folder and add the image.<br/>
        Then add the item to your <code>shared/items.lua</code>:
        ```lua
          ['revo_trading_tablet'] = {
            name = 'revo_trading_tablet',
            label = 'Trading Tablet',
            weight = 500,
            type = 'item',
            image = 'REVO_TRADING_TABLET.png',
            unique = false,
            useable = true,
            shouldClose = true,
            combinable = nil,
            description = 'Access trading panel.'
          }
        ```
        <Tip>
          Place <code>REVO_TRADING_TABLET.png</code> in <code>qb-inventory/html/images</code>.
        </Tip>
      </Tab>
    </Tabs>
  </Step>
  <Step title="Start the Resource">
    Restart your server or run these commands in the server console:
    ```bash
      refresh
      ensure revo_stocktrading
    ```
  </Step>
</Steps>

<Tip>
  Verify that <code>oxmysql</code> is connected and that all three SQL files were imported successfully before starting the resource.
  Use your MySQL client to check that the <code>stocks</code>, <code>trade_users</code>, and <code>trades</code> tables exist.
</Tip>
