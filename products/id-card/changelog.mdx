<Update label="April 2026" description="v1.2">
  Added a more flexible card system, fake ID validation improvements, and new inventory compatibility.

  - Added a config-driven card system so you can add or remove supported cards.
  - Added fake ID visual variations with randomized effects such as tint, spelling mistakes, blur on the player photo, and different date formats.
  - Added the police validation command `/idcheck` to help detect fake IDs.
  - Added configurable card display duration with `Config.CardDisplayDuration`.
  - Added card design selection per card with `design1`, `design2`, and `design3`.
  - Added qs-advancedinventory compatibility.
  - Updated export behavior so money deduction is optional and still defaults to enabled.
  - Updated default inventory image names to `revo_design_1.png` for `id_card`, `revo_design_2.png` for `driver_license`, and `revo_design_3.png` for `firearm_license`.
</Update>

<Update label="August 2025" description="v1.1">
  Added multi-inventory support and improved stability.

  - Added support for qs-inventory and qb-inventory systems.
  - Fixed QBCore-related bugs and compatibility issues.
  - Improved UI responsiveness and performance.
</Update>

<Update label="August 2025" description="v1.0">
  Initial release of Revo ID Card.

  - Real and fake ID card systems with configurable NPCs and map icons.
  - 5 unique card colors for different job types.
  - Separate designs for driver and firearm licenses.
  - Obtain cards from NPCs at custom locations.
  - Pricing options for each card type.
  - Beautiful UI with dark/light modes and card flipping.
  - Usable items for ox_inventory.
  - Show cards to nearby players with animation.
  - Multi-language support (10+ languages).
  - Fully optimized (0.00ms) and server-side secure.
  - Compatible with ESX, QBCore, and QBox.
</Update>
