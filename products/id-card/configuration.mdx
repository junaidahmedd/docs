title: 'Configuration'

<Info>
  Configuration is done in the <code>config.lua</code> file located in the <code>revo_idcard</code> directory.
</Info>

## Full Config Example

Use this as the current reference configuration:

```lua
Config = {}

Config.Theme = 'dark' -- Theme for menus, options: 'light', 'dark'
-- Available positions: 'top-left', 'top-right', 'top-center', 'bottom-left', 'bottom-right', 'bottom-center', center-left, center-right, center-center
Config.NotificationPosition = 'top-center' -- Default notification position

Config.UseFakeID = true -- Enable/Disable Fake ID feature
Config.Real_ID_NPC_Model = 'a_m_m_business_01' -- NPC Model for Real ID creation
Config.Real_ID_NPC_Locations = {
    {x = 435.098907, y = -974.769226, z = 30.712036, heading = 96.377945},
    {x = 214.298904, y = -947.367004, z = 30.678345, heading = 229.606293}
}
Config.Fake_ID_NPC_Model = 'g_m_y_ballaorig_01' -- NPC Model for Fake ID creation
Config.Fake_ID_NPC_Locations = {
    {x = 117.969231, y = -1951.582397, z = 20.737061, heading = 53.858269},
    {x = 155.024185, y = -1311.956055, z = 29.195557, heading = 48.188972}
}

Config.ShowRealIDMapIcon = true -- Show Real ID NPC on map
Config.RealIDMapIcon = {
    sprite = 480, -- Blip sprite for Real ID NPC
    scale = 0.9, -- Blip scale
    color = 2, -- Blip color
    name = 'ID Card' -- Blip name
}

Config.ShowFakeIDMapIcon = true -- Show Fake ID NPC on map
Config.FakeIDMapIcon = {
    sprite = 480, -- Blip sprite for Fake ID NPC
    scale = 0.9, -- Blip scale
    color = 1, -- Blip color
    name = 'Fake ID Dealer' -- Blip name
}

Config.EnableCardCosts = true -- Enable/Disable ID creation costs
Config.ShowIssueAndExpireDate = true -- Show issue and expiration date on ID
Config.ExpireDuration = 30 -- Expiration duration in days
Config.CardDisplayDuration = 4500 -- Card display duration in milliseconds
Config.ValidationCommand = 'idcheck' -- Police validation command
Config.ValidationCommandDistance = 3.0 -- Range for card validation command
Config.ValidationAllowedJobs = {
    police = true
}

-- Cooldown to prevent spam (in seconds)
Config.FakeIDCreationCooldown = 60 -- 1 minute

Config.StateNameOnCard = 'Los Santos' -- State name on ID card
Config.DefaultNationality = 'United States' -- Default Nationality
Config.IDCardViewDistance = 5.0 -- Distance to show ID card

Config.IDCardColors = {
    police = '#004dbb',
    ambulance = '#d32f2f',
    mechanic = '#ec9914',
    government = '#6a1b9a',
    taxi = '#f0c400',
    cardealer = '#0A3D62',
    unemployed = '#000000',
    default = '#000000'
}

-- Card types and their properties
-- design1 = default ID card layout
-- design2 = driver license layout
-- design3 = weapon license layout
Config.Cards = {
    ['id_card'] = {
       name = 'Identification Card',
       prop = 'prop_franklin_dl',
       design = 'design1',
       cost = 20,
       fakeCost = 100,
       enabled = true,
       purchasable = true
    },
    ['driver_license'] = {
       name = 'Driver License',
       prop = 'prop_franklin_dl',
       design = 'design2',
       cost = 50,
       enabled = true,
       purchasable = true
    },
    ['firearm_license'] = {
        name = 'Firearm License',
        prop = 'prop_franklin_dl',
        design = 'design3',
        cost = 100,
        enabled = true,
        purchasable = true
    },
}

-- Available languages: 'en', 'es', 'fr', 'de', 'it', 'pt', 'ru', 'pl', 'tr', 'zh', 'ar'
Config.DefaultLocale = 'en' -- Default language
```

## Theme

Set the menu theme:

```lua
Config.Theme = 'dark' -- Options: 'light', 'dark'
```

## Notification Position

Choose where notifications appear on the screen:

```lua
Config.NotificationPosition = 'top-center'
```

Available positions:  
`'top-left'`, `'top-right'`, `'top-center'`, `'bottom-left'`, `'bottom-right'`, `'bottom-center'`, `'center-left'`, `'center-right'`, `'center-center'`

## Fake ID Feature

Enable or disable the Fake ID system:

```lua
Config.UseFakeID = true
```

## NPC Configuration

Set NPC models and locations for real and fake ID creation:

```lua
Config.Real_ID_NPC_Model = 'a_m_m_business_01'
Config.Real_ID_NPC_Locations = {
    {x = 435.098907, y = -974.769226, z = 30.712036, heading = 96.377945},
    {x = 214.298904, y = -947.367004, z = 30.678345, heading = 229.606293}
}

Config.Fake_ID_NPC_Model = 'g_m_y_ballaorig_01'
Config.Fake_ID_NPC_Locations = {
    {x = 117.969231, y = -1951.582397, z = 20.737061, heading = 53.858269},
    {x = 155.024185, y = -1311.956055, z = 29.195557, heading = 48.188972}
}
```

## Map Icons

Show or hide map icons for NPCs:

```lua
Config.ShowRealIDMapIcon = true
Config.RealIDMapIcon = {
    sprite = 480,
    scale = 0.9,
    color = 2,
    name = 'ID Card'
}

Config.ShowFakeIDMapIcon = true
Config.FakeIDMapIcon = {
    sprite = 480,
    scale = 0.9,
    color = 1,
    name = 'Fake ID Dealer'
}
```

## Card Costs

Enable or disable card creation costs:

```lua
Config.EnableCardCosts = true
```

Each card's price is configured inside `Config.Cards` using `cost`, and fake ID pricing uses `fakeCost` where supported.

## Card Display & Validation

Control how cards are shown and how officers validate them:

```lua
Config.ShowIssueAndExpireDate = true
Config.ExpireDuration = 30
Config.CardDisplayDuration = 4500
Config.ValidationCommand = 'idcheck'
Config.ValidationCommandDistance = 3.0
Config.ValidationAllowedJobs = {
    police = true
}
```

## Cooldown

Set cooldown (in seconds) to prevent fake ID spam:

```lua
Config.FakeIDCreationCooldown = 60 -- 1 minute
```

## State & Nationality

Set the state name and default nationality shown on cards:

```lua
Config.StateNameOnCard = 'Los Santos'
Config.DefaultNationality = 'United States'
```

## Card View Distance

Set how far away players can see the ID card:

```lua
Config.IDCardViewDistance = 5.0
```

## Card Colors

Customize card colors for different job types:

```lua
Config.IDCardColors = {
    police = '#004dbb',
    ambulance = '#d32f2f',
    mechanic = '#ec9914',
    government = '#6a1b9a',
    taxi = '#f0c400',
    cardealer = '#0A3D62',
    unemployed = '#000000',
    default = '#000000'
}
```

## Card Types

Configure each card type and its properties:

```lua
Config.Cards = {
    ['id_card'] = {
       name = 'Identification Card',
       prop = 'prop_franklin_dl',
       design = 'design1',
       cost = 20,
       fakeCost = 100,
       enabled = true,
       purchasable = true
    },
    ['driver_license'] = {
       name = 'Driver License',
       prop = 'prop_franklin_dl',
       design = 'design2',
       cost = 50,
       enabled = true,
       purchasable = true
    },
    ['firearm_license'] = {
        name = 'Firearm License',
        prop = 'prop_franklin_dl',
        design = 'design3',
        cost = 100,
        enabled = true,
        purchasable = true
    },
}
```

Available designs:

- `design1`: Default ID card layout
- `design2`: Driver license layout
- `design3`: Weapon license layout

## Localization

Set the default language:

```lua
Config.DefaultLocale = 'en'
```

Supported languages: `'en'`, `'es'`, `'fr'`, `'de'`, `'it'`, `'pt'`, `'ru'`, `'pl'`, `'tr'`, `'zh'`, `'ar'`

<Tip>
  All features are fully configurable. Adjust NPCs, card visuals, validation rules, costs, and localization to fit your server's needs.
</Tip>
