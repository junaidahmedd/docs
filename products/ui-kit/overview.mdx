# Revo UI Kit

**Revo UI Kit** is a modern redesign of **ox_lib** and **ox_target**, built to give your server a cleaner interface and a more polished player experience. It focuses on modern styling, improved presentation, and customizable colors so you can better match your server branding.

## What It Does

- Redesigns the visual style of ox_lib-based interfaces
- Improves the look and feel of ox_target interactions
- Supports customizable colors for easier branding
- Keeps the familiar ox_lib / ox_target workflow while updating the design
