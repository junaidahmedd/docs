---
title: 'Installation'
description: 'Install Revo EcoWipe and set it up on your FiveM server.'
---

Follow these steps to install **Revo EcoWipe** in your ESX or QBCore server:

**Step 1**: Download or clone the script into your `resources` folder.

<CodeGroup>

bash GitHub (if available)
git clone https://github.com/revo-dev/ecowipe resources/[local]/ecowipe

bash Manual
Place the ecowipe folder inside `resources`

</CodeGroup>

**Step 2**: Add the resource to your `server.cfg`:

bash
ensure ecowipe

**Step 3**: Restart your server or run the following command from the console:

bash
refresh
ensure ecowipe

<Tip>
  EcoWipe is free and works for both online and offline players. All operations are performed in-game through a user-friendly UI.
</Tip>
